# Robotic Hand — No-Hardware Starter

Этот мини-набор позволяет без оборудования поиграться с пайплайном:
симулированный ЭМГ → фильтрация → распознавание (открыть/закрыть) → траектория "сервопривода".

## Установка
```bash
python -m venv .venv
# Windows: .venv\Scripts\activate
# Linux/Mac: source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск
1) Сгенерировать и визуализировать сигналы:
```bash
python emg_sim.py
```

2) Запустить простой цикл управления (с графиком):
```bash
python control_loop.py
```

## Что внутри
- `emg_sim.py` — создаёт синтетический ЭМГ: покой → активация → шум.
- `control_loop.py` — фильтры, огибающая, гистерезис, траектория "сервопривода" (0–180°).
- Логика легко переносится на Arduino/Teensy (сглаживание, мёртвые зоны, гистерезис).
